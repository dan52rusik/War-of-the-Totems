# War of the Totems

Стартовая структура проекта под прототип lane-battle с механикой тотемной эволюции.

## Папки

- `Scenes` - игровые и тестовые сцены.
- `Prefabs` - юниты, базы, UI-элементы и окружение.
- `Materials` - материалы для прототипа и финальной графики.
- `Textures` - временные и финальные текстуры.
- `Scripts/Runtime` - код игры.
- `Scripts/Editor` - инструменты редактора.

## Первый запуск

1. Открой Unity и дождись импорта ассетов.
2. В меню выбери `Tools > War of the Totems > Create Prototype Scene`.
3. Открой сцену `Assets/WarOfTheTotems/Scenes/PrototypeBattlefield.unity`.

## Что уже собирается

- Камера и базовый свет.
- Силуэты поля боя, двух баз и зоны эволюции.
- Заглушки юнитов: Base Bearer, Stone Guard и три Shadow Hunter.
- Верхняя и нижняя UI-панели с текстами из концепта.
